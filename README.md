## 使用：
1、下载，并复制CodeSnippets文件夹下的所有文件
<br>
2、在mac的finder中前往文件夹地址：
```
~/Library/Developer/Xcode/UserData/CodeSnippets
```
3、将第一步复制的内容，粘贴到第二步的地址下，重启Xcode
